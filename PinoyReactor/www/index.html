About Pinoy Reactor

PinoyReactor is a Javascript-based mobile game for Android, iOS and HTML5 Mobile Browsers.

The game highlights Filipino Culture and History as it's contents for this fast-paced quiz game.

Whoever gets the highest score through all the rounds wins the game.

Created by Good Game, Well Played (GGWP)

How to PLay

The game is easy to play! 5 random questions will be displayed from a random set.

Each question set has its own instructions.

The players must read the instructions and determine whether the value is correct.

If the question is correct, TAP the button. If not, do not tap it and wait for the next question.

Whoever the first player who has the highest point by the end of the game wins!
